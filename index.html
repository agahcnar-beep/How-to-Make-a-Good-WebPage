Learn HTML!

What does HTML even mean?

HTML a.k.a Hyper Text Mark-up Language is the base of a webpage. It helps us put text, images, audio and even references!

You can learn HTML by using:

  • W3schools
  • Khan Academy
  • Codecademy

Learn CSS!

What does CSS mean?

CSS a.k.a Cascading Style Sheets is the style of the webpage. It helps us decorate, position or make small transitions!

You can learn CSS by using:

  • Web.dev
  • Scrimba
  • Coursera

Learn JS!

What does JS mean?

JS a.k.a Java-Script is actually a coding language that has calculations unlike HTML and CSS, but in HTML references, it helps us create shapes, animations and transitions!

You can learn JS by using:

  • The Odin Project
  • freeCodeCamp
  • Mozilla Developer Network
